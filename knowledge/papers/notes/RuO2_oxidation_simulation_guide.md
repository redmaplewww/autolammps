# RuO2 氧化生长模拟文献综述与仿真设计指南

## 1. RuO2 晶体结构及晶面稳定性

### 1.1 基本晶体结构

- **结构类型**: 金红石 (rutile) 结构, 四方晶系
- **空间群**: P4_2/mnm (No. 136)
- **晶格常数**: a = b = 4.543 A, c = 3.140 A (Materials Project mp-825, DFT计算值)
- **密度**: 6.82 g/cm^3 (计算), 6.97 g/cm^3 (实验)
- **配位**: Ru^4+ 与六个 O^2- 形成畸变八面体, 包含 edge-sharing 和 corner-sharing
- **形成能**: -1.468 eV/atom (能量高于 hull = 0.000 eV, 稳定相)

### 1.2 常见晶面稳定性序列

基于金红石型氧化物的第一性原理研究, 各晶面稳定性排序为:

**(110) > (100) > (101) > (001) > (111)**

| 晶面 | 稳定性 | 表面特征 |
|------|--------|----------|
| (110) | 最稳定 | 包含桥氧 (O_br) 和配位不饱和Ru (Ru_cus), 是研究最多的面 |
| (100) | 较稳定 | 具有不同终止构型, 实验和DFT均有研究 |
| (101) | 中等 | 在薄膜外延生长中可被稳定化 |
| (001) | 高能 | 极性表面, 需要重构才能稳定 |
| (111) | 最高能 | 最少被研究 |

### 1.3 关键论文

- **Reuter & Scheffler (2001), PRB 65, 035406**: 证明 RuO2(110) 在低氧化学势下为化学计量比终止, 但在真实O2分压下会吸附额外顶位氧原子形成极性表面。传统离子模型判据对RuO2(110)无效。这是RuO2表面研究的基础性工作。

- **Reuter & Scheffler (2003), cond-mat/0301602**: DFT+热力学计算了RuO2(110)在O2和CO环境中的表面相图, 表明反应物吸附关键依赖温度和分压。

- **Sun, Reuter & Scheffler (2003), cond-mat/0302075**: 建立RuO2(110)在水蒸气中的相图, 微量氢足以在极低氧压下稳定富氧极性终止。

- **Kim et al. (2016), arXiv:1612.02890**: DFT研究苯在RuO2(110)的吸附, 确认缺O表面通过C-Ru键吸附强于富O表面通过H-O键吸附。

## 2. Ru金属 (hcp) 与 RuO2 (rutile) 之间的取向关系

### 2.1 Ru金属晶体结构

- **空间群**: P6_3/mmc (No. 194), 六方密堆积 (hcp)
- **晶格常数**: a = 2.706 A, c = 4.282 A (室温)
- **密度**: 12.45 g/cm^3

### 2.2 外延关系 (Epitaxial Relationships)

Ru(0001) 表面与 RuO2(110) 之间存在良好的晶格匹配:

- **Ru(0001) || RuO2(110)**: 这是最常报道的外延关系
  - Ru(0001) 面内六方对称 (a = 2.706 A)
  - RuO2(110) 表面具有矩形面内晶格 (c方向 = 3.14 A, [001]方向 = 4.54 A)
  - 晶格失配: ~14-16%, 但可通过旋转匹配减小

- **Ru(10-10) || RuO2(100)**: 另一潜在取向关系

**Key Reference**: 
- **Nunn et al. (2021), arXiv:2107.xxxxx**: 在TiO2(110)衬底上用固体源MOMBE外延生长了高质量RuO2薄膜, 在300摄氏度下达本体电阻率55 micro-ohm-cm。
- **Fischer et al. (2020), arXiv:2003.02723**: VO2在RuO2(110)和(100)上的MBE生长, RuO2(110)与VO2沿c轴有8.78%晶格失配, 实现完全应变生长; RuO2(100)上则完全弛豫。

## 3. Ru实验氧化速率

### 3.1 氧化动力学特征

Ru的氧化研究相对较少(相比Ni, Cu, Fe等常见金属), 已知的关键特征:

- **低温 (T < 400°C)**: Ru具有优良的抗氧化性, 氧化速率极慢。Ru薄膜在空气中7周后仅显示"minimal RuOx growth"(Langa et al. 2025)。

- **中高温 (T > 600°C)**: 氧化速率显著增加, 遵循抛物线氧化动力学:
  - 氧化层生长速率常数量级: ~10^(-14) 到 10^(-12) g^2/cm^4/s (700-1000°C)
  - 激活能估计范围: 120-200 kJ/mol (依赖温度范围和氧化层厚度)

- **高温 (T > 1000°C)**: RuO2 在约1200°C分解(熔点), 氧化层可能发生挥发。

- **高压 (T > 1100°C)**: Ru可作为保护涂层, 但抗氧化性不如Pt (Hosseinzadeh et al. 2025)。

### 3.2 氧化机理

- Ru的氧化遵循: Ru(s) + O2(g) -> RuO2(s)
- 初始阶段: O2在Ru表面的解离化学吸附
- 随后: O向内扩散 + Ru向外扩散, 形成连续RuO2层
- 在高温下: Ru + O2 -> RuO4(g) 挥发反应可能发生, 导致质量损失

## 4. Ru氧化的分子动力学模拟研究

### 4.1 现状总结

目前尚无专门针对Ru/RuO2氧化的ReaxFF反应力场MD模拟的公开报道。ReaxFF尚未有针对Ru-O体系的系统力场参数文件。

**替代方案**: 

**(a) 已存在ReaxFF力场的金属氧化先例:**
- Al/Al2O3 (已广泛验证, 是ReaxFF氧化的典型系统)
- Cu/Cu2O/CuO
- Si/SiO2
- Li-Al硅酸盐
- Ti(IV)-异丙氧化物分解

**(b) 如需要Ru-O ReaxFF参数, 有两种选择:**
1. 使用文献中已有RuO2力场参数(经验势, 非反应性): 需自行开发ReaxFF参数(训练集需要DFT数据)
2. 使用近似的过渡金属ReaxFF参数(风险: Ru与Fe/Ru有相似性但不精确)

### 4.2 相关MD氧化模拟的方法学参考

- **Lu et al. (2025)**: "Multi-AI Agent Framework Reveals the 'Oxide Gatekeeper' in Aluminum Nanoparticle Oxidation" 
  - 发展了Al氧化ReaxFF MD模拟的AI增强力场方法
  - 揭示了温度调节的双模式氧化机制: 低温"门控模式"(氧化物通过纳米通道呼吸) 和高温"破裂模式"(灾难性破裂导致爆炸燃烧)
  - 关键发现: Al阳离子向外扩散主导质量传输(与通常假设的氧向内扩散不同)

- **Yang et al. (2023)**: 反应分子动力学模拟显示O2分压控制氧化速率和形貌, 从逐层氧化到非晶氧化物介导的连续氧化转变。

## 5. 影响晶面依赖氧化速率的关键因素

| 因素 | 对氧化速率的影响 | 晶面依赖性 |
|------|-----------------|------------|
| **表面能** | 高能面反应活性高, 氧化更快 | (001) > (101) > (100) > (110) |
| **O2解离势垒** | 解离是速率控制步骤之一 | Ru(0001)上解离势垒 ~0.6-0.8 eV; 不同RuO2晶面有不同的活性位 |
| **O扩散** | O向内扩散是氧化层生长的速率控制步骤 (抛物线动力学) | 晶面取向影响扩散路径和势垒 |
| **外延应变** | 界面晶格失配产生应变, 影响扩散和反应势垒 | 不同Ru晶面与RuO2晶面的失配度不同 |
| **表面终止** | O_br vs. Ru_cus 比例影响反应性 | (110)上有明确的O_br和Ru_cus位; (100)终止不同 |

### 5.1 O2在Ru表面解离

- **O2在Ru(0001)上的解离**: DFT计算表明 O2 在清洁 Ru(0001) 表面可自发或极低势垒解离
- **O吸附结构**: 形成 p(2x2)-O 或 p(1x1)-O 覆盖层, 取决于温度和氧分压 (Stampfl & Scheffler 1996)
- **高氧覆盖效应**: 在高覆盖下 O-Ru 键强度减弱

## 6. 金属氧化模拟的典型方案

### 6.1 一般协议

| 参数 | 典型设置 |
|------|----------|
| **温度** | 300 - 1200 K (含温区覆盖整个氧化过程) |
| **升温速率** | NVT平衡后保持恒温 (NVT/NPT系综) |
| **O2压力** | 通过固定O2分子数目+体积估算; 或使用杂化NPT维持给定压强 |
| **模拟时间** | 0.1 - 10 ns (ReaxFF MD受限于计算成本) |
| **系综** | 初期 NVT (加热), 主体 NVE 或 NVT |
| **步长** | 0.1 - 0.25 fs (ReaxFF) 或 0.5 - 1 fs (经典势) |
| **周期性边界** | slab模型 3D周期性, z方向加真空层 |

### 6.2 模型构建

1. **Ru基底**: 构建 hcp Ru slab, 暴露目标晶面
   - (0001): 最常见, 密堆积面
   - (10-10): 棱柱面
   - (11-20): 第二棱柱面
2. **真空层**: 10-20 A
3. **O2层**: 在真空层中随机放置O2分子, 根据需要设置浓度
4. **底部固定**: 底部2-3层Ru原子固定

### 6.3 推荐温度方案

| 温度范围 | 目的 |
|----------|------|
| 300-600 K | 低温氧化, 观察初始O2吸附/解离 |
| 600-900 K | 中温氧化, 氧化层生长 |
| 900-1200 K | 高温氧化, 加速扩散过程 (用于跨时间尺度研究) |

## 7. 氧化层厚度的测量与定义

### 7.1 常用方法

| 方法 | 描述 | 适用性 |
|------|------|--------|
| **密度/浓度剖面** | 沿z方向计算密度或浓度分布, 定义RuO2区域边界 | 最常用, 易实现 |
| **序参数 (order parameter)** | 基于局部晶体或配位环境的序参数区分Ru和RuO2 | 需要定义参照结构 |
| **CNA/切向分析** | 对Ru原子做CNA分析, 识别hcp/非晶/FCC | 可区分金属和氧化区域 |
| **键序分析** | 基于Ru-O键的形成与否来标记"氧化"状态 | ReaxFF中自然适用 |
| **氧化Ru占比** | 每个Ru原子配位数>=2的O则标记为氧化态, 计算厚度 | 简单直观 |

### 7.2 推荐协议 (LAMMPS)

```
compute coord all coord/atom cutoff 1.5 3.0   # 第一近邻Ru-O距离~1.9-2.0 A
# 对每个Ru原子:
#   如果周围O原子数 >= 2 -> 标记为"氧化态Ru"
# 沿z方向分组统计, 找到氧化态Ru的分布边界
# 氧化层厚度 = 上边界 - 下边界
```

补充: 可使用密度剖面具体确定氧化物/金属界面位置, 取Ru密度从金属值下降到氧化值的中间位置。

## 8. 主要挑战

| 挑战 | 描述 | 缓解策略 |
|------|------|----------|
| **时间尺度** | MD可达 0.1-100 ns, 实验氧化在秒-小时尺度 | 升高温度加速; 使用加速MD (aMD, hyperdynamics); 或KMC |
| **力场精度** | 无公开Ru-O ReaxFF参数; EAM/MEAM势不支持反应 | 自训练ReaxFF; 使用COMB势 (Charge Optimized Many-Body); 或ML势 |
| **建模复杂度** | Ru(hcp) + RuO2(rutile) 界面结构不确定性 | 文献搜索取向关系; 构建共格/半共格界面 |
| **O扩散路径** | 金属中O扩散 vs 氧化物中O扩散 vs Ru向外扩散 | 需要示踪标记或能量计算确认 |
| **电荷转移** | 氧化涉及显著电荷转移, 传统势无法描述 | 使用ReaxFF (含电荷平衡) 或COMB势 |
| **挥发物形成** | RuO4(g) 在高温形成, 造成质量损失 | >1000K需特别关注, 涉及多体反应 |

## 9. 推荐仿真路线

### 第一阶段: 短时间/小规模探索

- **系统**: 小尺寸slab (500-2000原子), 单一晶面
- **力场**: 尝试EAM/MEAM (如果有Ru的金属势) + 自定义Ru-O势; 或直接开发Ru-O ReaxFF
- **温度**: 900-1200 K (加速扩散)
- **运行时间**: 0.5-2 ns
- **目标**: 验证初始氧化动力学, O2解离, 表面氧化物成核

### 第二阶段: 晶面比较

- **系统**: 多晶面 slab (110), (100), (001), 放大尺寸
- **力场**: 优化后的ReaxFF或COMB
- **温度**: 600-1200 K 逐步扫描
- **运行时间**: 1-5 ns
- **目标**: 比较不同晶面的氧化速率, 界面结构, 厚度-时间关系

### 第三阶段: 全长模拟

- **系统**: 大尺寸slab (10000+原子) 含晶界或缺陷, 多晶
- **力场**: 已验证的ReaxFF
- **温度**: 多温度点 (600, 800, 1000, 1200 K)
- **运行时间**: 5-20 ns (需要GPU加速)
- **目标**: 建立氧化动力学模型, 拟合抛物线速率常数

## 11. 注意事项

1. **力场选择是最关键限制**: 无公开的Ru-O ReaxFF参数, 需要从头训练或使用替代方法(COMB势, ML势)。
2. **实验数据有限**: Ru的氧化动力学研究比Ni/Cu/Fe少得多, 验证拟议的模拟方案会有困难。
3. **高温加速模拟需要谨慎**: 1000 K以上的模拟可能激活不同的扩散机理, 与实验温度不同。
4. **表面终止影响大**: Reuter & Scheffler (2001) 明确显示RuO2(110)的终止在不同氧化学势下高度可变, 模拟中需考虑初始终止态。
