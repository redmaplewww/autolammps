# Publication Figure Checklist

Before considering a figure final:

- [ ] Generated by a saved script.
- [ ] Input files and derived data are documented.
- [ ] Figure size matches intended use.
- [ ] PNG exported at 300+ DPI.
- [ ] Vector export exists for line-art plots.
- [ ] Axis labels include units where known.
- [ ] Text remains readable at final size.
- [ ] Legend is clear and does not cover data.
- [ ] Color palette is colorblind-safe.
- [ ] Figure works in grayscale or uses redundant encodings.
- [ ] Error bars are defined if present.
- [ ] Sample size/statistical test is shown or recorded if significance is shown.
- [ ] Multi-panel labels are consistent.
- [ ] No chart junk, 3D effects, or misleading truncated bar axes.
